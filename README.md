# Tailwind CSS - Edna Links

Proyecto web creado con HTML y Tailwind CSS.

## Descripción

Esta página muestra enlaces, cursos y redes sociales en un diseño tipo perfil de enlaces.

## Tecnologías utilizadas

- HTML5
- Tailwind CSS
- Node.js
- pnpm
